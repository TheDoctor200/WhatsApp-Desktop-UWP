## Useful Commands

### PowerShell Commands for Windows Apps

```powershell
# List all installed UWP apps
Get-AppxPackage | Select-Object Name, Version

# Get app details
Get-AppxPackage -Name "*WhatsApp*" | Format-List

# View app capabilities
Get-AppxPackage -Name "*WhatsApp*" | Get-AppxPackageManifest

# Check app installation location
Get-AppxPackage -Name "*WhatsApp*" | Select-Object InstallLocation

# Reset an app
Get-AppxPackage -Name "*WhatsApp*" | Reset-AppxPackage

# Export app info
Get-AppxPackage -Name "*WhatsApp*" | Export-Clixml -Path "app-info.xml"
```

### Windows System Information

```powershell
# Check Windows version
Get-ComputerInfo | Select-Object WindowsVersion, OSDisplayVersion

# Get system specifications
systeminfo

# Check .NET version
Get-ChildItem 'HKLM:\SOFTWARE\Microsoft\NET Framework Setup\NDP' -Recurse

# View Windows SDK versions
Get-ItemProperty 'HKLM:\SOFTWARE\WOW6432Node\Microsoft\Microsoft SDKs\Windows\*'
```
